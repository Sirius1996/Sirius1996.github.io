Semi-supervised Node Classification via Hierarchical Graph Convolutional Networks 笔记

Mar 20, 2019

图卷积网络（GCN）已成功应用于网络挖掘的节点分类任务。 然而，基于邻域聚合的这些模型中的大多数通常是浅的并且缺乏“图池化”机制，这阻止了模型获得足够的全局信息。 为了增加感受野，我们提出了一种新的深层次图解卷积网络（H-GCN）用于半监督节点分类。

Abstract

图卷积网络（GCN）已成功应用于网络挖掘的节点分类任务。 然而，基于邻域聚合的这些模型中的大多数通常是浅的并且缺乏“图池化”机制，这阻止了模型获得足够的全局信息。 为了增加感受野，我们提出了一种新的深层次图解卷积网络（H-GCN）用于半监督节点分类。

H-GCN首先重复地将结构相似的节点聚合到超节点，然后将粗化的图形细化为原始图形以恢复每个节点的表示。 建议的粗化过程不是仅仅聚合一个或两个邻域信息，而是扩大了每个节点的接收场，因此可以学习更多的全局信息。

Introduction

基于GCN的模型通常很浅，缺乏"graph pooling"机制，这限制了接受的规模。GCN只有两层，由于每个图卷积层充当一阶邻域上的聚合的近似，因此两层GCN模型仅聚合来自每个节点的两跳邻域的信息。由于接受的规模有限，模型难以接收足够的全局信息。

本文提出了一种新颖的分层图卷积网络体系结构，简化为H-GCN，用于图上的节点分类。设计了一个具有粗化机制的深层次模型。如下图所示。

对于每个粗化层，首先进行图卷积运算以学习节点表示。然后，执行粗化操作以将结构上类似的节点聚合成超节点。每个超节点表示原始图的一个局部结构，这有助于利用图上的全局结构。在粗化层之后，我们应用对称图形细化层来恢复节点分类任务的原始图形结构。
